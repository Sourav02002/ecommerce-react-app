E-Commerce App

A full-featured e-commerce web application built with React.js.

## Features
- JWT-based user authentication
- Protected routes
- Product listing with search and filters
- Product detail page
- Cart functionality with Context API
- Responsive design

## Tech Stack
- React.js
- React Router
- Context API
- REST API integration
- CSS
